# 171.Excel表列序号
####题目：
给定一个Excel表格中的列名称，返回其相应的列序号。

例如，

    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28
    ...

示例 1:

输入: "A"
输出: 1

#####自己的代码
```java
class Solution {
    public int titleToNumber(String s) {
        int sum= 0 ;
        for(int i =0; i<s.length(); i++) {
            char a = s.charAt(i);  
            sum = sum*26+(int)(a-64);
        }
        return sum;
    }
}
```

class Solution {
    public int titleToNumber(String s) {
        char[]charArray=s.toCharArray();
        int num=0;

        for(int i=charArray.length-1;i>=0;i--)
        {
        	num+=(int)(charArray[i]-64)*x;

        }
